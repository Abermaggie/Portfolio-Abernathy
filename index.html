About Me

Hi, I'm Maggie Abernathy! I'm studying to be a web developer/coder in the Vanderbilt coding bootcamp. For me, the decision to enter this bootcamp was a bit of a gamble because I wasn't sure what topic would truly inspire me.

For one of my first homework projects, I was instructed to create my own coding portfolio online so that potential employers could access my content and finished projects to get a feel for the skills I bring to the table within the coding areana.

This website is the fruit of that labor and I hope to add many interesting projects as I begin to grow and learn more about the type of coding that is uniquely specific to my interests.
